# Andrea Pozzato — CV

Welcome! This is my personal resume/CV, served as a web application. It provides a brutalist-styled frontend with my work experience, skills, projects, and contact information.

The easiest way to view my CV is by visiting the live site or running the Docker container:

```bash
docker pull pzzt/cv:latest
docker run -p 8000:8000 pzzt/cv:latest
# Open http://localhost:8000 in your browser
```
